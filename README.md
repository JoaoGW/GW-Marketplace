## 📦 Instalação

Siga os passos abaixo para rodar o projeto localmente:

No bash: <br/>
### Clone o repositório
git clone https://github.com/JoaoGW/GuiaTuristico.git

### Acesse o diretório
cd GW-Marketplace

### Instale as dependências
npm install
ou
yarn install

### Inicie a aplicação
npx expo start
